## Key Points

- stem:[x^0 = 1] and stem:[x^{-m} = \frac{1}{x^m}]
- stem:[x^{\frac{1}{n}} = \sqrt[n\]{x}]
- stem:[x^m \times x^n = x^{m+n}]
- stem:[x^m \div x^n = x^{m-n}]
